# LLM 学习知识库

系统化的大语言模型学习仓库，面向希望补齐认知、理解原理、再逐步上手实践的中文读者。

这不是新闻流仓库，也不是“追模型榜单”的速报仓库。它更适合用来建立一套相对稳定、不过时的 LLM 知识框架，再在局部章节里补充新变化。

## 这套知识库适合谁

- 想系统理解 LLM，而不是只会调用 API 的读者
- 想补齐 Transformer、预训练、微调、推理、RAG、Agent 等核心概念的读者
- 想从“会用模型”走向“能判断方案、能读懂实现、能搭基础应用”的读者

## 各部分解决什么问题

### 01-基础入门

解决“我知道 LLM 很火，但到底是什么、该怎么开始学”的问题。

### 02-核心原理

解决“我会用模型，但不清楚它为什么这样工作”的问题。

### 03-动手实践

解决“我想把概念落到代码和训练流程里”的问题。

### 04-进阶应用

解决“我想把模型接进真实应用场景”的问题。

### 05-实战项目

解决“我需要一个完整项目视角，而不是零散知识点”的问题。

### 06-主流开源模型

解决“不同模型家族各自适合什么场景、技术路线有什么差异”的问题。

### 07-MCP 协议

解决“模型和工具、外部系统如何更稳定地连接”的问题。

## 学习建议

### 如果你是初学者

- 先建立概念图谱，再进代码细节。
- 不要一开始就陷入模型排行榜和参数规模对比。
- 先搞清楚：token、上下文窗口、预训练、微调、推理、RAG、Agent 分别是什么。

### 如果你已经在做项目

- 把重点放在推理、评测、RAG、Agent 和开源模型选型上。
- 少做“盲目追新”，多做“方案判断”。

### 如果你想跟进变化

- 把“稳定认知”和“短期热点”分开。
- 先读本仓库的框架性内容，再去追模型版本更新。
