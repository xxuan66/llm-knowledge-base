# LLM 学习知识库

欢迎来到系统化的大语言模型（LLM）学习知识库！本知识库整合了 GitHub 热门资源，构建从基础到进阶的中文学习体系，帮助你逐步掌握 LLM 的核心技术。

## 学习路径

本知识库分为六个阶段，循序渐进：

### 📚 第一阶段：基础入门
了解 LLM 的基本概念、发展历程和核心术语，建立整体认知框架。

### 🧠 第二阶段：核心原理
深入理解 Transformer 架构、注意力机制、预训练与微调等核心技术。

### ⚙️ 第三阶段：动手实践
从零开始搭建 LLM、实现 Tokenizer、体验预训练流程、掌握微调与量化技术。

### 🚀 第四阶段：进阶应用
学习 RAG 检索增强、Agent 智能体、模型评测和前沿技术。

### 🛠️ 第五阶段：实战项目
完成对话机器人、知识库问答、多 Agent 协作、微调专属模型等实战项目。

### 📖 第六阶段：主流模型
深入学习 Llama、Qwen、Mistral、DeepSeek、GLM 等主流开源模型的架构细节和部署方法。
