# 微调与量化

## 学习目标
学完本节后，你将能够：
- 掌握 LLM 微调的核心技术和流程
- 理解参数高效微调（PEFT）方法
- 了解模型量化的原理和实现
- 能够在实际项目中应用微调和量化技术

## 核心知识点

### 1. 微调技术

#### 1.1 全参数微调
- **方法**：更新模型的所有参数
- **优点**：性能最大化，适合数据充足的场景
- **缺点**：计算成本高，存储需求大
- **适用**：拥有大量标注数据和计算资源

#### 1.2 参数高效微调（PEFT）
**核心思想**：只更新少量参数，冻结大部分模型

**主要方法**：
1. **LoRA（Low-Rank Adaptation）**
   - 添加低秩适配器
   - 参数量：减少 99%+
   - 存储：每个任务只需几 MB

2. **Adapter**
   - 在 Transformer 层中插入小型模块
   - 参数量：减少 90%+
   - 适配器：小的前馈网络

3. **Prompt Tuning**
   - 优化输入提示词
   - 参数量：最少
   - 适用：提示敏感的任务

4. **Prefix Tuning**
   - 优化前缀表示
   - 参数量：中等
   - 适用：生成任务

#### 1.3 指令微调
- **目标**：让模型学会遵循指令
- **数据**：指令-输出对
- **代表**：InstructGPT、ChatGPT

### 2. 量化技术

#### 2.1 量化基础
- **定义**：降低模型参数精度，减少内存占用
- **目标**：在保持性能的同时减少资源需求
- **精度转换**：FP32 → FP16/BF16 → INT8 → INT4

#### 2.2 量化方法
**2.2.1 训练后量化（PTQ）**
- **流程**：训练 → 量化
- **优点**：简单，无需重新训练
- **缺点**：可能损失精度

**2.2.2 量化感知训练（QAT）**
- **流程**：在训练中模拟量化
- **优点**：精度损失小
- **缺点**：需要重新训练

**2.2.3 4-bit 量化**
- **方法**：GPTQ、AWQ
- **优点**：内存减少 4 倍
- **缺点**：需要校准数据

#### 2.3 量化格式
- **INT8**：8 位整数，平衡精度和速度
- **INT4**：4 位整数，大幅减少内存
- **NF4**：4 位归一化浮点，保持精度
- **FP4**：4 位浮点，最新研究方向

### 3. 微调 vs 量化对比

| 维度 | 微调 | 量化 |
|------|------|------|
| **目标** | 提高特定任务性能 | 减少资源需求 |
| **方法** | 参数更新 | 精度降低 |
| **计算成本** | 中等到高 | 低 |
| **存储需求** | 增加 | 减少 |
| **适用场景** | 任务适配 | 部署优化 |

### 4. 结合使用策略

#### 4.1 LoRA + 量化
- **流程**：先量化基础模型，再添加 LoRA 适配器
- **优点**：节省存储和计算
- **适用**：资源受限的部署场景

#### 4.2 QLoRA
- **方法**：4-bit 量化基础模型 + LoRA 微调
- **创新**：双重量化，进一步减少内存
- **效果**：可在单 GPU 上微调大模型

## 代码示例

### LoRA 微调实现

```python
import torch
import torch.nn as nn
from transformers import AutoModelForCausalLM, AutoTokenizer
from peft import LoraConfig, get_peft_model, TaskType
from datasets import load_dataset
from transformers import TrainingArguments, Trainer

class LoRAFineTuner:
    """LoRA 微调器"""
    
    def __init__(self, model_name, lora_rank=8, lora_alpha=16, target_modules=None):
        self.model_name = model_name
        self.lora_rank = lora_rank
        self.lora_alpha = lora_alpha
        self.target_modules = target_modules or ["q_proj", "v_proj"]
        
        # 加载 tokenizer
        self.tokenizer = AutoTokenizer.from_pretrained(model_name)
        if self.tokenizer.pad_token is None:
            self.tokenizer.pad_token = self.tokenizer.eos_token
        
        # 加载基础模型
        self.base_model = AutoModelForCausalLM.from_pretrained(
            model_name,
            torch_dtype=torch.float16,
            device_map="auto"
        )
        
        # 配置 LoRA
        self.lora_config = LoraConfig(
            r=lora_rank,
            lora_alpha=lora_alpha,
            target_modules=self.target_modules,
            lora_dropout=0.1,
            bias="none",
            task_type=TaskType.CAUSAL_LM
        )
        
        # 应用 LoRA
        self.model = get_peft_model(self.base_model, self.lora_config)
        self.model.print_trainable_parameters()
    
    def prepare_dataset(self, dataset_name, split="train", max_length=512):
        """准备数据集"""
        dataset = load_dataset(dataset_name, split=split)
        
        def tokenize_function(examples):
            # 格式化指令数据
            if "instruction" in examples and "output" in examples:
                texts = []
                for instruction, output in zip(examples["instruction"], examples["output"]):
                    text = f"指令：{instruction}\n输出：{output}"
                    texts.append(text)
            else:
                texts = examples["text"]
            
            # 分词
            tokenized = self.tokenizer(
                texts,
                truncation=True,
                max_length=max_length,
                padding="max_length",
                return_tensors="pt"
            )
            
            # 创建标签（输入部分设为 -100）
            labels = tokenized["input_ids"].clone()
            labels[tokenized["attention_mask"] == 0] = -100
            
            return {
                "input_ids": tokenized["input_ids"],
                "attention_mask": tokenized["attention_mask"],
                "labels": labels
            }
        
        return dataset.map(tokenize_function, batched=True, remove_columns=dataset.column_names)
    
    def train(self, train_dataset, val_dataset=None, 
              output_dir="./lora-finetuned", num_epochs=3, batch_size=4):
        """训练模型"""
        
        training_args = TrainingArguments(
            output_dir=output_dir,
            num_train_epochs=num_epochs,
            per_device_train_batch_size=batch_size,
            per_device_eval_batch_size=batch_size,
            gradient_accumulation_steps=4,
            evaluation_strategy="steps" if val_dataset else "no",
            eval_steps=100 if val_dataset else None,
            save_strategy="steps",
            save_steps=100,
            save_total_limit=3,
            learning_rate=2e-4,
            fp16=True,
            logging_steps=10,
            optim="adamw_torch",
            warmup_ratio=0.03,
            lr_scheduler_type="cosine",
            report_to="none"
        )
        
        trainer = Trainer(
            model=self.model,
            args=training_args,
            train_dataset=train_dataset,
            eval_dataset=val_dataset,
            tokenizer=self.tokenizer,
        )
        
        # 训练
        trainer.train()
        
        # 保存适配器
        self.model.save_pretrained(output_dir)
        self.tokenizer.save_pretrained(output_dir)
        
        print(f"模型已保存到 {output_dir}")
    
    def inference(self, prompt, max_length=100):
        """推理"""
        self.model.eval()
        
        # 编码输入
        inputs = self.tokenizer(prompt, return_tensors="pt").to(self.model.device)
        
        # 生成
        with torch.no_grad():
            outputs = self.model.generate(
                **inputs,
                max_length=max_length,
                temperature=0.7,
                do_sample=True,
                pad_token_id=self.tokenizer.pad_token_id
            )
        
        # 解码
        generated_text = self.tokenizer.decode(outputs[0], skip_special_tokens=True)
        return generated_text

# 使用示例
if __name__ == "__main__":
    # 加载微调器
```

### 4-bit 量化实现

```python
import torch
from transformers import AutoModelForCausalLM, AutoTokenizer, BitsAndBytesConfig
from peft import prepare_model_for_kbit_training, LoraConfig, get_peft_model

class QuantizedModel:
    """4-bit 量化模型"""
    
    def __init__(self, model_name, load_in_4bit=True, use_double_quant=True):
        self.model_name = model_name
        
        # 量化配置
        self.bnb_config = BitsAndBytesConfig(
            load_in_4bit=load_in_4bit,
            bnb_4bit_use_double_quant=use_double_quant,
            bnb_4bit_quant_type="nf4",
            bnb_4bit_compute_dtype=torch.float16
        )
        
        # 加载量化模型
        self.model = AutoModelForCausalLM.from_pretrained(
            model_name,
            quantization_config=self.bnb_config,
            device_map="auto",
            torch_dtype=torch.float16
        )
        
        # 加载 tokenizer
        self.tokenizer = AutoTokenizer.from_pretrained(model_name)
        if self.tokenizer.pad_token is None:
            self.tokenizer.pad_token = self.tokenizer.eos_token
        
        # 准备模型用于训练
        self.model = prepare_model_for_kbit_training(self.model)
    
    def add_lora(self, lora_rank=8):
        """添加 LoRA 适配器"""
        lora_config = LoraConfig(
            r=lora_rank,
            lora_alpha=16,
            target_modules=["q_proj", "v_proj", "k_proj", "o_proj"],
            lora_dropout=0.1,
            bias="none",
            task_type="CAUSAL_LM"
        )
        
        self.model = get_peft_model(self.model, lora_config)
        self.model.print_trainable_parameters()
        return self.model
    
    def get_memory_usage(self):
        """获取内存使用情况"""
        memory_stats = {}
        
        # 模型参数内存
        param_memory = sum(p.numel() * p.element_size() for p in self.model.parameters())
        memory_stats['parameters_mb'] = param_memory / (1024 ** 2)
        
        # 梯度内存
        grad_memory = sum(p.numel() * p.element_size() for p in self.model.parameters() if p.requires_grad)
        memory_stats['gradients_mb'] = grad_memory / (1024 ** 2)
        
        # 激活内存（近似）
        memory_stats['activations_mb'] = 100  # 示例值
        
        # 总内存
        memory_stats['total_mb'] = sum(memory_stats.values())
        
        return memory_stats
    
    def inference(self, prompt, max_length=100):
        """推理"""
        self.model.eval()
        
        inputs = self.tokenizer(prompt, return_tensors="pt").to(self.model.device)
        
        with torch.no_grad():
            outputs = self.model.generate(
                **inputs,
                max_length=max_length,
                temperature=0.7,
                do_sample=True,
                pad_token_id=self.tokenizer.pad_token_id
            )
        
        return self.tokenizer.decode(outputs[0], skip_special_tokens=True)

# 使用示例
if __name__ == "__main__":
    # 加载量化模型
    quantized = QuantizedModel(
        model_name="gpt2",  # 使用 GPT-2 作为示例
        load_in_4bit=True,
        use_double_quant=True
    )
    
    # 添加 LoRA
    model = quantized.add_lora(lora_rank=8)
    
    # 查看内存使用
    memory = quantized.get_memory_usage()
    print("内存使用情况:")
    for key, value in memory.items():
        print(f"  {key}: {value:.2f} MB")
    
    # 测试推理
    prompt = "机器学习是"
    result = quantized.inference(prompt, max_length=50)
    print(f"\n生成结果: {result}")
```

### QLoRA 微调完整流程

```python
import torch
from transformers import (
    AutoModelForCausalLM,
    AutoTokenizer,
    BitsAndBytesConfig,
    TrainingArguments
)
from peft import LoraConfig, get_peft_model, prepare_model_for_kbit_training
from trl import SFTTrainer
from datasets import load_dataset

class QLoRATrainer:
    """QLoRA 训练器"""
    
    def __init__(self, model_name, output_dir="./qlora-output"):
        self.model_name = model_name
        self.output_dir = output_dir
        
        # 量化配置
        self.bnb_config = BitsAndBytesConfig(
            load_in_4bit=True,
            bnb_4bit_use_double_quant=True,
            bnb_4bit_quant_type="nf4",
            bnb_4bit_compute_dtype=torch.float16
        )
        
        # 加载 tokenizer
        self.tokenizer = AutoTokenizer.from_pretrained(model_name)
        self.tokenizer.pad_token = self.tokenizer.eos_token
        self.tokenizer.padding_side = "right"
        
        # 加载量化模型
        self.model = AutoModelForCausalLM.from_pretrained(
            model_name,
            quantization_config=self.bnb_config,
            device_map="auto"
        )
        
        # 准备模型
        self.model = prepare_model_for_kbit_training(self.model)
        
        # LoRA 配置
        self.lora_config = LoraConfig(
            r=16,
            lora_alpha=32,
            target_modules=["q_proj", "k_proj", "v_proj", "o_proj"],
            lora_dropout=0.05,
            bias="none",
            task_type="CAUSAL_LM"
        )
        
        # 应用 LoRA
        self.model = get_peft_model(self.model, self.lora_config)
        self.model.print_trainable_parameters()
    
    def format_instruction(self, example):
        """格式化指令数据"""
        if "instruction" in example and "output" in example:
            if "input" in example and example["input"]:
                text = f"### 指令：{example['instruction']}\n### 输入：{example['input']}\n### 输出：{example['output']}"
            else:
                text = f"### 指令：{example['instruction']}\n### 输出：{example['output']}"
        else:
            text = example["text"]
        
        return {"text": text}
    
    def train(self, dataset_name="tatsu-lab/alpaca", max_seq_length=512, num_epochs=3):
        """训练模型"""
        # 加载数据集
        dataset = load_dataset(dataset_name, split="train")
        
        # 格式化数据集
        dataset = dataset.map(
            self.format_instruction,
            remove_columns=dataset.column_names
        )
        
        # 训练参数
        training_args = TrainingArguments(
            output_dir=self.output_dir,
            num_train_epochs=num_epochs,
            per_device_train_batch_size=4,
            gradient_accumulation_steps=4,
            optim="paged_adamw_32bit",
            save_steps=100,
            logging_steps=10,
            learning_rate=2e-4,
            fp16=True,
            max_grad_norm=0.3,
            warmup_ratio=0.03,
            lr_scheduler_type="cosine",
            report_to="none"
        )
        
        # 创建 SFT Trainer
        trainer = SFTTrainer(
            model=self.model,
            train_dataset=dataset,
            tokenizer=self.tokenizer,
            args=training_args,
            max_seq_length=max_seq_length,
            dataset_text_field="text",
            packing=False
        )
        
        # 训练
        trainer.train()
        
        # 保存模型
        trainer.save_model()
        self.tokenizer.save_pretrained(self.output_dir)
        
        print(f"模型已保存到 {self.output_dir}")
    
    def merge_and_save(self, output_path):
        """合并 LoRA 适配器并保存"""
        # 合并权重
        merged_model = self.model.merge_and_unload()
        
        # 保存完整模型
        merged_model.save_pretrained(output_path)
        self.tokenizer.save_pretrained(output_path)
        
        print(f"合并后的模型已保存到 {output_path}")

# 使用示例
if __name__ == "__main__":

    trainer = QLoRATrainer(
        model_name="gpt2",  # 使用 GPT-2 作为示例
        output_dir="./qlora-finetuned"
    )
```
