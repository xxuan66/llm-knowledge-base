# 微调与量化

## 学习目标

学完本节后，你应该能够：

- 知道微调和量化分别解决什么问题
- 理解为什么今天项目里更常见的是 PEFT、LoRA、QLoRA
- 在部署与适配之间做更现实的取舍判断
- 避免把“该做 RAG 的问题”误判成“必须微调”

## 先说一个常见误区

很多人接触 LLM 工程后，会把“效果不够好”第一时间归因到“是不是要微调”。

但现实中更常见的正确顺序是：

1. 先优化提示词和结构化输出
2. 再判断是否需要 RAG
3. 再考虑工具调用或工作流约束
4. 最后才考虑是否真的需要微调

微调和量化都很重要，但它们解决的是不同问题：

- 微调：让模型行为更贴合目标任务
- 量化：让模型更容易跑起来、更省资源

## 1. 微调到底在改什么

### 1.1 全参数微调

全参数微调会更新模型的全部参数。优点是上限高，但代价也最大：

- 算力需求高
- 训练时间长
- 存储多个版本成本高
- 运维复杂度更高

### 1.2 参数高效微调（PEFT）

PEFT 的核心思路是：冻结大部分原模型，只训练少量新增或可调参数。

这也是今天大多数业务团队更现实的选择。

**主要方法**：
1. **LoRA（Low-Rank Adaptation）**
   - 添加低秩适配器
   - 参数量：减少 99%+
   - 存储：每个任务只需几 MB

2. **Adapter**
   - 在 Transformer 层中插入小型模块
   - 参数量：减少 90%+
   - 适配器：小的前馈网络

3. **Prompt Tuning**
   - 优化输入提示词
   - 参数量：最少
   - 适用：提示敏感的任务

4. **Prefix Tuning**
   - 优化前缀表示
   - 参数量：中等
   - 适用：生成任务

### 1.3 指令微调

指令微调会显著影响模型“像不像一个能用的助手”。很多场景里，模型从“会续写”变成“会执行要求”，关键就发生在这里。

## 2. 量化是在解决部署问题

### 2.1 量化基础

量化的核心不是“让模型更聪明”，而是让模型更省显存、更容易部署、更适合在线推理。

### 2.2 量化方法
**2.2.1 训练后量化（PTQ）**
- **流程**：训练 → 量化
- **优点**：简单，无需重新训练
- **缺点**：可能损失精度

**2.2.2 量化感知训练（QAT）**
- **流程**：在训练中模拟量化
- **优点**：精度损失小
- **缺点**：需要重新训练

**2.2.3 4-bit 量化**
- **方法**：GPTQ、AWQ
- **优点**：内存减少 4 倍
- **缺点**：需要校准数据

### 2.3 量化格式

不同格式的核心取舍是：

- 资源占用
- 精度损失
- 推理速度
- 框架支持情况

## 3. 微调和量化不要混为一谈

| 维度 | 微调 | 量化 |
|------|------|------|
| **目标** | 提高特定任务性能 | 减少资源需求 |
| **方法** | 参数更新 | 精度降低 |
| **计算成本** | 中等到高 | 低 |
| **存储需求** | 增加 | 减少 |
| **适用场景** | 任务适配 | 部署优化 |

## 4. 最常见的实战组合

### 4.1 LoRA + 量化

这是最常见也最现实的工程组合之一：

- 用量化降低基础模型运行成本
- 用 LoRA 做轻量任务适配

### 4.2 QLoRA

QLoRA 之所以重要，不是因为名字新，而是因为它把“在有限资源上调大模型”这件事变得现实可行。

## 5. 一个更实用的判断框架

遇到任务效果不好时，可以先这样判断：

### 先问自己三个问题

1. 问题是知识不够新，还是行为不够稳定？
2. 问题是回答不准，还是回答格式不符合要求？
3. 问题是模型不会做，还是系统没把信息喂对？

### 一般经验

- 知识问题：先看 RAG
- 格式问题：先看提示词和约束输出
- 领域语气 / 行为模式问题：再看微调
- 成本问题：看量化

## 一个常见结论

很多团队真正需要的不是“重新训练模型”，而是：

- 先选合适底模
- 先把提示词和上下文结构设计好
- 先把检索和工具接起来
- 最后只在必要时做轻量微调

### LoRA 微调实现

```python
import torch
import torch.nn as nn
from transformers import AutoModelForCausalLM, AutoTokenizer
from peft import LoraConfig, get_peft_model, TaskType
from datasets import load_dataset
from transformers import TrainingArguments, Trainer

class LoRAFineTuner:
    """LoRA 微调器"""
    
    def __init__(self, model_name, lora_rank=8, lora_alpha=16, target_modules=None):
        self.model_name = model_name
        self.lora_rank = lora_rank
        self.lora_alpha = lora_alpha
        self.target_modules = target_modules or ["q_proj", "v_proj"]
        
        # 加载 tokenizer
        self.tokenizer = AutoTokenizer.from_pretrained(model_name)
        if self.tokenizer.pad_token is None:
            self.tokenizer.pad_token = self.tokenizer.eos_token
        
        # 加载基础模型
        self.base_model = AutoModelForCausalLM.from_pretrained(
            model_name,
            torch_dtype=torch.float16,
            device_map="auto"
        )
        
        # 配置 LoRA
        self.lora_config = LoraConfig(
            r=lora_rank,
            lora_alpha=lora_alpha,
            target_modules=self.target_modules,
            lora_dropout=0.1,
            bias="none",
            task_type=TaskType.CAUSAL_LM
        )
        
        # 应用 LoRA
        self.model = get_peft_model(self.base_model, self.lora_config)
        self.model.print_trainable_parameters()
    
    def prepare_dataset(self, dataset_name, split="train", max_length=512):
        """准备数据集"""
        dataset = load_dataset(dataset_name, split=split)
        
        def tokenize_function(examples):
            # 格式化指令数据
            if "instruction" in examples and "output" in examples:
                texts = []
                for instruction, output in zip(examples["instruction"], examples["output"]):
                    text = f"指令：{instruction}\n输出：{output}"
                    texts.append(text)
            else:
                texts = examples["text"]
            
            # 分词
            tokenized = self.tokenizer(
                texts,
                truncation=True,
                max_length=max_length,
                padding="max_length",
                return_tensors="pt"
            )
            
            # 创建标签（输入部分设为 -100）
            labels = tokenized["input_ids"].clone()
            labels[tokenized["attention_mask"] == 0] = -100
            
            return {
                "input_ids": tokenized["input_ids"],
                "attention_mask": tokenized["attention_mask"],
                "labels": labels
            }
        
        return dataset.map(tokenize_function, batched=True, remove_columns=dataset.column_names)
    
    def train(self, train_dataset, val_dataset=None, 
              output_dir="./lora-finetuned", num_epochs=3, batch_size=4):
        """训练模型"""
        
        training_args = TrainingArguments(
            output_dir=output_dir,
            num_train_epochs=num_epochs,
            per_device_train_batch_size=batch_size,
            per_device_eval_batch_size=batch_size,
            gradient_accumulation_steps=4,
            evaluation_strategy="steps" if val_dataset else "no",
            eval_steps=100 if val_dataset else None,
            save_strategy="steps",
            save_steps=100,
            save_total_limit=3,
            learning_rate=2e-4,
            fp16=True,
            logging_steps=10,
            optim="adamw_torch",
            warmup_ratio=0.03,
            lr_scheduler_type="cosine",
            report_to="none"
        )
        
        trainer = Trainer(
            model=self.model,
            args=training_args,
            train_dataset=train_dataset,
            eval_dataset=val_dataset,
            tokenizer=self.tokenizer,
        )
        
        # 训练
        trainer.train()
        
        # 保存适配器
        self.model.save_pretrained(output_dir)
        self.tokenizer.save_pretrained(output_dir)
        
        print(f"模型已保存到 {output_dir}")
    
    def inference(self, prompt, max_length=100):
        """推理"""
        self.model.eval()
        
        # 编码输入
        inputs = self.tokenizer(prompt, return_tensors="pt").to(self.model.device)
        
        # 生成
        with torch.no_grad():
            outputs = self.model.generate(
                **inputs,
                max_length=max_length,
                temperature=0.7,
                do_sample=True,
                pad_token_id=self.tokenizer.pad_token_id
            )
        
        # 解码
        generated_text = self.tokenizer.decode(outputs[0], skip_special_tokens=True)
        return generated_text

# 使用示例
```

### 4-bit 量化实现

```python
import torch
from transformers import AutoModelForCausalLM, AutoTokenizer, BitsAndBytesConfig
from peft import prepare_model_for_kbit_training, LoraConfig, get_peft_model

class QuantizedModel:
    """4-bit 量化模型"""
    
    def __init__(self, model_name, load_in_4bit=True, use_double_quant=True):
        self.model_name = model_name
        
        # 量化配置
        self.bnb_config = BitsAndBytesConfig(
            load_in_4bit=load_in_4bit,
            bnb_4bit_use_double_quant=use_double_quant,
            bnb_4bit_quant_type="nf4",
            bnb_4bit_compute_dtype=torch.float16
        )
        
        # 加载量化模型
        self.model = AutoModelForCausalLM.from_pretrained(
            model_name,
            quantization_config=self.bnb_config,
            device_map="auto",
            torch_dtype=torch.float16
        )
        
        # 加载 tokenizer
        self.tokenizer = AutoTokenizer.from_pretrained(model_name)
        if self.tokenizer.pad_token is None:
            self.tokenizer.pad_token = self.tokenizer.eos_token
        
        # 准备模型用于训练
        self.model = prepare_model_for_kbit_training(self.model)
    
    def add_lora(self, lora_rank=8):
        """添加 LoRA 适配器"""
        lora_config = LoraConfig(
            r=lora_rank,
            lora_alpha=16,
            target_modules=["q_proj", "v_proj", "k_proj", "o_proj"],
            lora_dropout=0.1,
            bias="none",
            task_type="CAUSAL_LM"
        )
        
        self.model = get_peft_model(self.model, lora_config)
        self.model.print_trainable_parameters()
        return self.model
    
    def get_memory_usage(self):
        """获取内存使用情况"""
        memory_stats = {}
        
        # 模型参数内存
        param_memory = sum(p.numel() * p.element_size() for p in self.model.parameters())
        memory_stats['parameters_mb'] = param_memory / (1024 ** 2)
        
        # 梯度内存
        grad_memory = sum(p.numel() * p.element_size() for p in self.model.parameters() if p.requires_grad)
        memory_stats['gradients_mb'] = grad_memory / (1024 ** 2)
        
        # 激活内存（近似）
        memory_stats['activations_mb'] = 100  # 示例值
        
        # 总内存
        memory_stats['total_mb'] = sum(memory_stats.values())
        
        return memory_stats
    
    def inference(self, prompt, max_length=100):
        """推理"""
        self.model.eval()
        
        inputs = self.tokenizer(prompt, return_tensors="pt").to(self.model.device)
        
        with torch.no_grad():
            outputs = self.model.generate(
                **inputs,
                max_length=max_length,
                temperature=0.7,
                do_sample=True,
                pad_token_id=self.tokenizer.pad_token_id
            )
        
        return self.tokenizer.decode(outputs[0], skip_special_tokens=True)

# 使用示例
if __name__ == "__main__":
    # 加载量化模型
    quantized = QuantizedModel(
        model_name="gpt2",  # 使用 GPT-2 作为示例
        load_in_4bit=True,
        use_double_quant=True
    )
    
    # 添加 LoRA
    model = quantized.add_lora(lora_rank=8)
    
    # 查看内存使用
    memory = quantized.get_memory_usage()
    print("内存使用情况:")
    for key, value in memory.items():
        print(f"  {key}: {value:.2f} MB")
    
    # 测试推理
    prompt = "机器学习是"
    result = quantized.inference(prompt, max_length=50)
    print(f"\n生成结果: {result}")
```

### QLoRA 微调完整流程

```python
import torch
from transformers import (
    AutoModelForCausalLM,
    AutoTokenizer,
    BitsAndBytesConfig,
    TrainingArguments
)
from peft import LoraConfig, get_peft_model, prepare_model_for_kbit_training
from trl import SFTTrainer
from datasets import load_dataset

class QLoRATrainer:
    """QLoRA 训练器"""
    
    def __init__(self, model_name, output_dir="./qlora-output"):
        self.model_name = model_name
        self.output_dir = output_dir
        
        # 量化配置
        self.bnb_config = BitsAndBytesConfig(
            load_in_4bit=True,
            bnb_4bit_use_double_quant=True,
            bnb_4bit_quant_type="nf4",
            bnb_4bit_compute_dtype=torch.float16
        )
        
        # 加载 tokenizer
        self.tokenizer = AutoTokenizer.from_pretrained(model_name)
        self.tokenizer.pad_token = self.tokenizer.eos_token
        self.tokenizer.padding_side = "right"
        
        # 加载量化模型
        self.model = AutoModelForCausalLM.from_pretrained(
            model_name,
            quantization_config=self.bnb_config,
            device_map="auto"
        )
        
        # 准备模型
        self.model = prepare_model_for_kbit_training(self.model)
        
        # LoRA 配置
        self.lora_config = LoraConfig(
            r=16,
            lora_alpha=32,
            target_modules=["q_proj", "k_proj", "v_proj", "o_proj"],
            lora_dropout=0.05,
            bias="none",
            task_type="CAUSAL_LM"
        )
        
        # 应用 LoRA
        self.model = get_peft_model(self.model, self.lora_config)
        self.model.print_trainable_parameters()
    
    def format_instruction(self, example):
        """格式化指令数据"""
        if "instruction" in example and "output" in example:
            if "input" in example and example["input"]:
                text = f"### 指令：{example['instruction']}\n### 输入：{example['input']}\n### 输出：{example['output']}"
            else:
                text = f"### 指令：{example['instruction']}\n### 输出：{example['output']}"
        else:
            text = example["text"]
        
        return {"text": text}
    
    def train(self, dataset_name="tatsu-lab/alpaca", max_seq_length=512, num_epochs=3):
        """训练模型"""
        # 加载数据集
        dataset = load_dataset(dataset_name, split="train")
        
        # 格式化数据集
        dataset = dataset.map(
            self.format_instruction,
            remove_columns=dataset.column_names
        )
        
        # 训练参数
        training_args = TrainingArguments(
            output_dir=self.output_dir,
            num_train_epochs=num_epochs,
            per_device_train_batch_size=4,
            gradient_accumulation_steps=4,
            optim="paged_adamw_32bit",
            save_steps=100,
            logging_steps=10,
            learning_rate=2e-4,
            fp16=True,
            max_grad_norm=0.3,
            warmup_ratio=0.03,
            lr_scheduler_type="cosine",
            report_to="none"
        )
        
        # 创建 SFT Trainer
        trainer = SFTTrainer(
            model=self.model,
            train_dataset=dataset,
            tokenizer=self.tokenizer,
            args=training_args,
            max_seq_length=max_seq_length,
            dataset_text_field="text",
            packing=False
        )
        
        # 训练
        trainer.train()
        
        # 保存模型
        trainer.save_model()
        self.tokenizer.save_pretrained(self.output_dir)
        
        print(f"模型已保存到 {self.output_dir}")
    
    def merge_and_save(self, output_path):
        """合并 LoRA 适配器并保存"""
        # 合并权重
        merged_model = self.model.merge_and_unload()
        
        # 保存完整模型
        merged_model.save_pretrained(output_path)
        self.tokenizer.save_pretrained(output_path)
        
        print(f"合并后的模型已保存到 {output_path}")
```

## 完整微调工作流示例

### 1. 端到端微调工作流

```python
"""
完整微调工作流实现
包含数据准备、训练、评估、部署全流程
"""
import os
import json
from typing import Dict, List, Optional
from dataclasses import dataclass
from pathlib import Path
import torch
from transformers import (
    AutoModelForCausalLM,
    AutoTokenizer,
    TrainingArguments,
    Trainer,
    DataCollatorForLanguageModeling
)
from peft import LoraConfig, get_peft_model, TaskType
from datasets import Dataset, load_dataset
import wandb

@dataclass
class FineTuningConfig:
    """微调配置"""
    model_name: str
    output_dir: str
    lora_rank: int = 8
    lora_alpha: int = 16
    lora_dropout: float = 0.1
    target_modules: List[str] = None
    learning_rate: float = 2e-4
    num_epochs: int = 3
    batch_size: int = 4
    gradient_accumulation_steps: int = 4
    max_seq_length: int = 512
    warmup_steps: int = 100
    logging_steps: int = 10
    save_steps: int = 100
    eval_steps: int = 100
    
    def __post_init__(self):
        if self.target_modules is None:
            self.target_modules = ["q_proj", "v_proj", "k_proj", "o_proj"]

class DataProcessor:
    """数据处理器"""
    
    def __init__(self, tokenizer, max_length: int = 512):
        self.tokenizer = tokenizer
        self.max_length = max_length
    
    def load_data(self, data_path: str) -> List[Dict]:
        """加载数据"""
        path = Path(data_path)
        
        if path.suffix == '.json':
            with open(path) as f:
                return json.load(f)
        elif path.suffix == '.jsonl':
            data = []
            with open(path) as f:
                for line in f:
                    data.append(json.loads(line))
            return data
        elif path.suffix == '.csv':
            import pandas as pd
            df = pd.read_csv(path)
            return df.to_dict('records')
        else:
            raise ValueError(f"不支持的文件格式: {path.suffix}")
    
    def format_instruction(self, example: Dict) -> str:
        """格式化指令数据"""
        if "instruction" in example and "output" in example:
            instruction = example["instruction"]
            input_text = example.get("input", "")
            output = example["output"]
            
            if input_text:
                return f"### 指令:\n{instruction}\n\n### 输入:\n{input_text}\n\n### 回答:\n{output}"
            else:
                return f"### 指令:\n{instruction}\n\n### 回答:\n{output}"
        elif "messages" in example:
            # 对话格式
            messages = example["messages"]
            formatted = []
            for msg in messages:
                role = msg["role"]
                content = msg["content"]
                formatted.append(f"{role}: {content}")
            return "\n".join(formatted)
        else:
            return example.get("text", str(example))
    
    def tokenize_function(self, examples: Dict) -> Dict:
        """分词函数"""
        # 格式化文本
        texts = []
        for i in range(len(examples[list(examples.keys())[0]])):
            example = {k: examples[k][i] for k in examples.keys()}
            texts.append(self.format_instruction(example))
        
        # 分词
        tokenized = self.tokenizer(
            texts,
            truncation=True,
            max_length=self.max_length,
            padding="max_length",
            return_tensors="pt"
        )
        
        # 创建标签
        labels = tokenized["input_ids"].clone()
        labels[tokenized["attention_mask"] == 0] = -100
        
        return {
            "input_ids": tokenized["input_ids"],
            "attention_mask": tokenized["attention_mask"],
            "labels": labels
        }
    
    def prepare_dataset(self, data_path: str) -> Dataset:
        """准备数据集"""
        # 加载数据
        data = self.load_data(data_path)
        
        # 转换为 Dataset
        dataset = Dataset.from_list(data)
        
        # 分词
        tokenized_dataset = dataset.map(
            self.tokenize_function,
            batched=True,
            remove_columns=dataset.column_names
        )
        
        return tokenized_dataset

class FineTuningWorkflow:
    """完整微调工作流"""
    
    def __init__(self, config: FineTuningConfig):
        self.config = config
        self.model = None
        self.tokenizer = None
        self.trainer = None
        
        # 初始化 wandb
        wandb.init(
            project="llm-finetuning",
            name=f"finetune-{config.model_name}",
            config=config.__dict__
        )
    
    def setup_model(self):
        """设置模型和分词器"""
        print(f"加载模型: {self.config.model_name}")
        
        # 加载分词器
        self.tokenizer = AutoTokenizer.from_pretrained(self.config.model_name)
        if self.tokenizer.pad_token is None:
            self.tokenizer.pad_token = self.tokenizer.eos_token
        
        # 加载模型
        self.model = AutoModelForCausalLM.from_pretrained(
            self.config.model_name,
            torch_dtype=torch.float16,
            device_map="auto",
            load_in_4bit=True,  # 使用 4-bit 量化
            bnb_4bit_compute_dtype=torch.float16,
            bnb_4bit_quant_type="nf4"
        )
        
        # 配置 LoRA
        lora_config = LoraConfig(
            r=self.config.lora_rank,
            lora_alpha=self.config.lora_alpha,
            target_modules=self.config.target_modules,
            lora_dropout=self.config.lora_dropout,
            bias="none",
            task_type=TaskType.CAUSAL_LM
        )
        
        # 应用 LoRA
        self.model = get_peft_model(self.model, lora_config)
        self.model.print_trainable_parameters()
    
    def prepare_data(self, train_path: str, val_path: str = None):
        """准备数据"""
        print("准备训练数据...")
        
        processor = DataProcessor(self.tokenizer, self.config.max_seq_length)
        
        # 训练数据
        train_dataset = processor.prepare_dataset(train_path)
        print(f"训练样本数: {len(train_dataset)}")
        
        # 验证数据
        val_dataset = None
        if val_path:
            val_dataset = processor.prepare_dataset(val_path)
            print(f"验证样本数: {len(val_dataset)}")
        
        return train_dataset, val_dataset
    
    def train(self, train_dataset: Dataset, val_dataset: Dataset = None):
        """训练模型"""
        print("开始训练...")
        
        # 训练参数
        training_args = TrainingArguments(
            output_dir=self.config.output_dir,
            num_train_epochs=self.config.num_epochs,
            per_device_train_batch_size=self.config.batch_size,
            per_device_eval_batch_size=self.config.batch_size,
            gradient_accumulation_steps=self.config.gradient_accumulation_steps,
            evaluation_strategy="steps" if val_dataset else "no",
            eval_steps=self.config.eval_steps if val_dataset else None,
            save_strategy="steps",
            save_steps=self.config.save_steps,
            save_total_limit=3,
            learning_rate=self.config.learning_rate,
            warmup_steps=self.config.warmup_steps,
            logging_steps=self.config.logging_steps,
            fp16=True,
            bf16=False,
            optim="paged_adamw_8bit",
            lr_scheduler_type="cosine",
            report_to="wandb",
            load_best_model_at_end=True if val_dataset else False,
            metric_for_best_model="eval_loss" if val_dataset else None,
        )
        
        # 数据整理器
        data_collator = DataCollatorForLanguageModeling(
            tokenizer=self.tokenizer,
            mlm=False
        )
        
        # 创建 Trainer
        self.trainer = Trainer(
            model=self.model,
            args=training_args,
            train_dataset=train_dataset,
            eval_dataset=val_dataset,
            data_collator=data_collator,
        )
        
        # 训练
        train_result = self.trainer.train()
        
        # 保存模型
        self.trainer.save_model()
        self.tokenizer.save_pretrained(self.config.output_dir)
        
        print(f"训练完成！模型保存到: {self.config.output_dir}")
        print(f"训练结果: {train_result}")
    
    def evaluate(self, test_dataset: Dataset) -> Dict:
        """评估模型"""
        print("评估模型...")
        
        if self.trainer is None:
            raise ValueError("模型尚未训练")
        
        # 评估
        eval_results = self.trainer.evaluate(test_dataset)
        
        print(f"评估结果: {eval_results}")
        return eval_results
    
    def merge_and_save(self, output_path: str):
        """合并 LoRA 权重并保存"""
        print(f"合并权重并保存到: {output_path}")
        
        # 合并权重
        merged_model = self.model.merge_and_unload()
        
        # 保存
        merged_model.save_pretrained(output_path)
        self.tokenizer.save_pretrained(output_path)
        
        print("合并完成！")
    
    def export_to_gguf(self, output_path: str, quantization: str = "q4_k_m"):
        """导出为 GGUF 格式"""
        print(f"导出为 GGUF 格式: {output_path}")
        
        # 需要 llama.cpp 的 convert.py 脚本
        import subprocess
        
        cmd = [
            "python", "convert.py",
            self.config.output_dir,
            "--outtype", quantization,
            "--outfile", output_path
        ]
        
        subprocess.run(cmd, check=True)
        print(f"GGUF 导出完成: {output_path}")

# 使用示例
```

### 3. 微调工作流自动化脚本

**文件：** `finetuning_workflow_automation.py`

```python
"""
微调工作流自动化脚本
支持数据准备、训练、评估、部署全流程
"""
import os
import json
from pathlib import Path
from datetime import datetime
import yaml

class FineTuningAutomation:
    """微调工作流自动化"""
    
    def __init__(self, config_path: str):
        self.config = self.load_config(config_path)
        self.workflow = None
    
    def load_config(self, config_path: str) -> dict:
        """加载配置"""
        with open(config_path) as f:
            return yaml.safe_load(f)
    
    def prepare_data(self):
        """准备数据"""
        print("准备训练数据...")
        
        # 创建数据目录
        data_dir = Path(self.config["data"]["train_path"]).parent
        data_dir.mkdir(parents=True, exist_ok=True)
        
        # 检查数据文件
        train_path = Path(self.config["data"]["train_path"])
        if not train_path.exists():
            print(f"警告: 训练数据文件不存在: {train_path}")
            print("请准备训练数据文件")
            return False
        
        return True
    
    def run_training(self):
        """运行训练"""
        print("开始训练...")
        
        # 创建配置
        from finetuning_workflow import FineTuningConfig, FineTuningWorkflow
        
        config = FineTuningConfig(
            model_name=self.config["model"]["name"],
            output_dir=self.config["output"]["dir"],
            lora_rank=self.config["lora"]["rank"],
            lora_alpha=self.config["lora"]["alpha"],
            learning_rate=self.config["training"]["learning_rate"],
            num_epochs=self.config["training"]["epochs"],
            batch_size=self.config["training"]["batch_size"],
            max_seq_length=self.config["data"]["preprocessing"]["max_length"]
        )
        
        # 创建工作流
        self.workflow = FineTuningWorkflow(config)
        
        # 设置模型
        self.workflow.setup_model()
        
        # 准备数据
        train_dataset, val_dataset = self.workflow.prepare_data(
            train_path=self.config["data"]["train_path"],
            val_path=self.config["data"].get("val_path")
        )
        
        # 训练
        self.workflow.train(train_dataset, val_dataset)
        
        # 合并权重
        if self.config["output"]["merge_weights"]["enabled"]:
            self.workflow.merge_and_save(
                self.config["output"]["merge_weights"]["output_dir"]
            )
        
        # 导出 GGUF
        if self.config["output"]["export"]["gguf"]["enabled"]:
            self.workflow.export_to_gguf(
                self.config["output"]["export"]["gguf"]["output_path"],
                self.config["output"]["export"]["gguf"]["quantization"]
            )
        
        print("训练完成！")
    
    def run_evaluation(self):
        """运行评估"""
        print("运行评估...")
        
        if self.workflow is None:
            print("错误: 工作流未初始化")
            return
        
        # 加载测试数据
        test_path = self.config["data"].get("test_path")
        if test_path and Path(test_path).exists():
            from finetuning_workflow import DataProcessor
            
            processor = DataProcessor(
                self.workflow.tokenizer,
                self.config["data"]["preprocessing"]["max_length"]
            )
            test_dataset = processor.prepare_dataset(test_path)
            
            # 评估
            results = self.workflow.evaluate(test_dataset)
            
            # 保存评估结果
            eval_dir = Path("./evaluation")
            eval_dir.mkdir(exist_ok=True)
            
            eval_file = eval_dir / f"eval_{datetime.now().strftime('%Y%m%d_%H%M%S')}.json"
            with open(eval_file, 'w') as f:
                json.dump(results, f, indent=2)
            
            print(f"评估结果已保存到: {eval_file}")
    
    def run_full_workflow(self):
        """运行完整工作流"""
        print("=" * 50)
        print("开始微调工作流")
        print("=" * 50)
        
        # 1. 准备数据
        if not self.prepare_data():
            return
        
        # 2. 训练
        self.run_training()
        
        # 3. 评估
        self.run_evaluation()
        
        print("=" * 50)
        print("微调工作流完成！")
        print("=" * 50)

# 使用示例
if __name__ == "__main__":
    automation = FineTuningAutomation("finetuning_workflow_config.yaml")
    automation.run_full_workflow()
```
