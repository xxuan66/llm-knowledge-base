# 其他国产开源模型

> 除 Qwen、DeepSeek、GLM 外，中国还有多家机构开源了优秀的大语言模型，包括 Yi（零一万物）、InternLM（商汤）、Baichuan（百川）、MiniMax 等。

这一页的重点不是逐个追版本，而是帮你快速判断：在主流大厂模型之外，还有哪些路线值得纳入比较范围。

## 怎么使用这一页

建议把这页当作补充候选池，而不是主线入口。

适合在这些情况下重点参考：

- 你已经比较过 Qwen / DeepSeek / GLM
- 你想找更适合本地部署或长上下文的替代路线
- 你想了解国产生态里不同团队的技术取向

## 一、Yi 系列（零一万物）

### 1.1 模型概览

| 模型 | 发布时间 | 参数量 | 上下文 | 特点 |
|------|---------|--------|--------|------|
| Yi-6B | 2023.11 | 6B | 4K | 轻量高效 |
| Yi-34B | 2023.11 | 34B | 4K | 高性能 |
| Yi-34B-200K | 2024.01 | 34B | 200K | 超长上下文 |
| Yi-Large | 2024.05 | undisclosed | 128K | 旗舰模型 |

### 1.2 架构特点

**Yi-34B 架构参数：**
- 参数量：34B
- 层数：60 层
- 注意力头数：64 Q 头 / 8 KV 头（GQA）
- 隐藏层维度：7168
- 中间层维度（FFN）：20480
- 词表大小：64,000

**核心技术：**
1. **GQA 注意力** — 高效推理
2. **高质量语料** — 精选中英文数据
3. **长上下文优化** — 200K 版本支持超长文档

### 1.3 性能表现

| 基准 | Yi-34B | Llama2-70B | Qwen-72B |
|------|--------|-----------|---------|
| MMLU | 75.9 | 68.9 | 77.4 |
| GSM8K | 73.5 | 58.4 | 80.2 |
| HumanEval | 58.5 | 46.3 | 72.0 |

### 1.4 使用示例

```python
from transformers import AutoModelForCausalLM, AutoTokenizer

model_name = "01-ai/Yi-34B-Chat"
tokenizer = AutoTokenizer.from_pretrained(model_name)
model = AutoModelForCausalLM.from_pretrained(
    model_name,
    torch_dtype=torch.bfloat16,
    device_map="auto"
)

# 对话格式
messages = [
    {"role": "user", "content": "你好，介绍一下你自己"}
]

text = tokenizer.apply_chat_template(messages, tokenize=False)
inputs = tokenizer(text, return_tensors="pt").to(model.device)

outputs = model.generate(**inputs, max_new_tokens=512)
print(tokenizer.decode(outputs[0], skip_special_tokens=True))
```

## 二、InternLM 系列（商汤）

### 2.1 模型概览

| 模型 | 发布时间 | 参数量 | 上下文 | 特点 |
|------|---------|--------|--------|------|
| InternLM-7B | 2023.07 | 7B | 8K | 基座模型 |
| InternLM-20B | 2023.09 | 20B | 8K | 高性能 |
| InternLM2-7B | 2024.01 | 7B | 32K | 架构升级 |
| InternLM2.5-7B | 2024.06 | 7B | 128K | 全面增强 |

### 2.2 架构特点

**InternLM2-7B 架构参数：**
- 参数量：7B
- 层数：32 层
- 注意力头数：32 Q 头 / 8 KV 头（GQA）
- 隐藏层维度：4096
- 中间层维度（FFN）：11008
- 词表大小：103,168

**核心技术：**
1. **混合注意力** — 结合全局和局部注意力
2. **高稀疏度 MoE** — 部分版本采用 MoE
3. **长上下文优化** — 128K 支持

### 2.3 工具调用能力

InternLM2 强化了工具调用能力：

```python
from transformers import AutoModel, AutoTokenizer

tokenizer = AutoTokenizer.from_pretrained("internlm/internlm2-chat-7b", trust_remote_code=True)
model = AutoModel.from_pretrained("internlm/internlm2-chat-7b", trust_remote_code=True).cuda()

# 工具定义
tools = [
    {"type": "function", "function": {"name": "search", "description": "搜索信息"}}
]

# 对话
response, history = model.chat(
    tokenizer,
    "帮我查一下今天的新闻",
    history=[],
    tools=tools
)

print(response)
```

## 三、Baichuan 系列（百川）

### 3.1 模型概览

| 模型 | 发布时间 | 参数量 | 上下文 | 特点 |
|------|---------|--------|--------|------|
| Baichuan-7B | 2023.06 | 7B | 4K | 初代版本 |
| Baichuan2-7B | 2023.09 | 7B | 32K | 性能提升 |
| Baichuan2-13B | 2023.09 | 13B | 32K | 更大规模 |

### 3.2 架构特点

**Baichuan2-13B 架构参数：**
- 参数量：13B
- 层数：40 层
- 注意力头数：40 Q 头 / 5 KV 头（GQA）
- 隐藏层维度：5120
- 中间层维度（FFN）：13824
- 词表大小：125,696

**核心技术：**
1. **GQA 注意力** — 高效推理
2. **中文优化** — 针对中文场景优化
3. **医疗/法律垂直** — 有专业领域版本

### 3.3 使用示例

```python
from transformers import AutoModelForCausalLM, AutoTokenizer

tokenizer = AutoTokenizer.from_pretrained("baichuan-inc/Baichuan2-7B-Chat", trust_remote_code=True)
model = AutoModelForCausalLM.from_pretrained(
    "baichuan-inc/Baichuan2-7B-Chat",
    torch_dtype=torch.float16,
    trust_remote_code=True,
    device_map="auto"
)

messages = [{"role": "user", "content": "你好"}]
response = model.chat(tokenizer, messages)
print(response)
```

## 一个实用判断

这些模型未必都应该成为主力选择，但很适合作为：

- 备选方案
- 特定场景补充方案
- 横向比较时的参照对象

## 六、选择建议

### 6.1 按场景选择

| 场景 | 推荐模型 | 理由 |
|------|---------|------|
| 本地部署 | Yi-6B / InternLM2-7B | 轻量高效 |
| 长文档 | Yi-34B-200K | 200K 上下文 |
| 中文任务 | Baichuan2 / GLM-4 | 中文优化 |
| 工具调用 | InternLM2 / ChatGLM3 | 原生支持 |
| 企业级 | Yi-Large / GLM-4 | 最佳性能 |

### 6.2 按硬件选择

| 硬件 | 推荐模型 |
|------|---------|
| RTX 3060 12G | Yi-6B, InternLM2-7B, Baichuan2-7B |
| RTX 3090 24G | Yi-34B(INT4), Baichuan2-13B(INT4) |
| 2×A100 80G | Yi-34B, InternLM2-20B, Qwen2-32B |
| 4×A100 80G | Qwen2-72B, DeepSeek-V2 |

## 七、部署资源

### 7.1 Ollama 支持

```bash
# Yi 系列
ollama run yi:6b
ollama run yi:34b

# InternLM 系列
ollama run internlm2:7b

# Baichuan 系列（需手动导入）
ollama create baichuan2 -f ./Modelfile
```

### 7.2 vLLM 部署

```bash
# Yi
python -m vllm.entrypoints.api_server --model 01-ai/Yi-34B-Chat

# InternLM
python -m vllm.entrypoints.api_server --model internlm/internlm2-chat-7b --trust-remote-code

# Baichuan
python -m vllm.entrypoints.api_server --model baichuan-inc/Baichuan2-7B-Chat --trust-remote-code
```
