# 模型评测

## 学习目标
学完本节后，你将能够：
- 理解 LLM 评测的核心概念和方法
- 掌握常见评测基准和指标
- 了解模型评测的实践流程
- 能够设计和实施模型评测方案

## 核心知识点

### 1. 评测基础

#### 1.1 为什么需要评测
- **模型选择**：比较不同模型的性能
- **能力评估**：了解模型在特定任务上的表现
- **改进方向**：发现模型的弱点和改进空间
- **安全评估**：检测模型的偏见和安全问题

#### 1.2 评测维度
1. **知识能力**：事实性知识、常识推理
2. **推理能力**：逻辑推理、数学推理
3. **语言能力**：语法、语义、风格
4. **安全性**：偏见、有害内容、鲁棒性
5. **效率**：推理速度、资源消耗

#### 1.3 评测方法分类
- **自动评测**：使用指标自动计算
- **人工评测**：人类评估员打分
- **混合评测**：两者结合

### 2. 常见评测基准

#### 2.1 综合能力评测
**MMLU（Massive Multitask Language Understanding）**
- **任务**：57 个学科的选择题
- **领域**：STEM、人文、社会科学等
- **特点**：综合性强，广泛使用

**AGIEval**
- **任务**：人类考试题目（高考、SAT 等）
- **特点**：更贴近人类评估标准
- **优势**：可解释性好

#### 2.2 推理能力评测
**GSM8K（Grade School Math 8K）**
- **任务**：小学数学应用题
- **特点**：需要多步推理
- **评估**：准确率

**HumanEval**
- **任务**：代码生成
- **特点**：评估编程能力
- **指标**：pass@k

**DROP（Discrete Reasoning Over Paragraphs）**
- **任务**：阅读理解推理
- **特点**：需要离散推理
- **评估**：F1 分数

#### 2.3 安全性评测
**TruthfulQA**
- **任务**：评估模型真实性
- **特点**：检测幻觉和虚假信息
- **指标**：真实性比例

**RealToxicityPrompts**
- **任务**：检测毒性内容生成
- **特点**：评估安全性
- **指标**：毒性分数

**WinoGender**
- **任务**：性别偏见检测
- **特点**：评估偏见问题
- **指标**：偏见程度

#### 2.4 中文评测
**C-Eval**
- **任务**：中文综合评测
- **特点**：52 个学科，中英双语
- **评估**：准确率

**CMMLU**
- **任务**：中文多任务理解
- **特点**：67 个任务
- **评估**：综合评分

### 3. 评测指标

#### 3.1 自动指标
**准确率（Accuracy）**
- 正确预测数 / 总预测数
- 适用于分类任务

**F1 分数**
- 精确率和召回率的调和平均
- 适用于信息检索、NER

**BLEU/ROUGE**
- 文本生成质量评估
- BLEU：机器翻译
- ROUGE：文本摘要

**Pass@k**
- 代码生成评估
- k 次尝试中至少一次通过

#### 3.2 人工指标
**Likert 量表**
- 1-5 分或 1-7 分评分
- 评估流畅性、相关性等

**偏好比较**
- A/B 测试
- 成对比较

**错误分析**
- 分类错误类型
- 分析错误原因

### 4. 评测实践

#### 4.1 评测流程
1. **确定评测目标**：明确要评估什么
2. **选择评测基准**：选择合适的基准和数据
3. **设计评测方案**：自动/人工/混合
4. **实施评测**：运行评测代码
5. **分析结果**：统计分析和可视化
6. **报告撰写**：总结发现和建议

#### 4.2 评测工具
- **lm-evaluation-harness**：通用评测框架
- **OpenCompass**：中文评测工具
- **Helm**：Stanford 的评测框架
- **FastChat**：聊天模型评测

#### 4.3 注意事项
- **数据污染**：避免训练数据泄漏
- **过拟合**：防止针对特定基准优化
- **公平性**：确保评测条件一致
- **可复现性**：记录评测环境和参数

## 代码示例

### 自动评测框架

```python
import json
import numpy as np
from typing import List, Dict, Any
from dataclasses import dataclass
import matplotlib.pyplot as plt
from collections import defaultdict

@dataclass
class EvaluationResult:
    """评测结果"""
    model_name: str
    benchmark: str
    metrics: Dict[str, float]
    details: Dict[str, Any] = None

class ModelEvaluator:
    """模型评测器"""
    
    def __init__(self):
        self.results = []
    
    def evaluate_accuracy(self, predictions: List[int], labels: List[int]) -> Dict[str, float]:
        """计算准确率相关指标"""
        predictions = np.array(predictions)
        labels = np.array(labels)
        
        # 准确率
        accuracy = np.mean(predictions == labels)
        
        # 精确率、召回率、F1（针对二分类）
        if len(np.unique(labels)) == 2:
            tp = np.sum((predictions == 1) & (labels == 1))
            fp = np.sum((predictions == 1) & (labels == 0))
            fn = np.sum((predictions == 0) & (labels == 1))
            
            precision = tp / (tp + fp) if (tp + fp) > 0 else 0
            recall = tp / (tp + fn) if (tp + fn) > 0 else 0
            f1 = 2 * precision * recall / (precision + recall) if (precision + recall) > 0 else 0
            
            return {
                'accuracy': accuracy,
                'precision': precision,
                'recall': recall,
                'f1': f1
            }
        
        return {'accuracy': accuracy}
    
    def evaluate_generation(self, references: List[str], hypotheses: List[str]) -> Dict[str, float]:
        """评估文本生成质量"""
        # 简化的 BLEU 计算（实际应使用 sacrebleu 或 nltk）
        from collections import Counter
        
        def get_ngrams(text, n=2):
            words = text.lower().split()
            return [tuple(words[i:i+n]) for i in range(len(words)-n+1)]
        
        # BLEU-2 简化计算
        bleu_scores = []
        for ref, hyp in zip(references, hypotheses):
            ref_ngrams = set(get_ngrams(ref, 2))
            hyp_ngrams = get_ngrams(hyp, 2)
            
            if not hyp_ngrams:
                bleu_scores.append(0)
                continue
            
            # 计算匹配的 n-gram 数量
            matches = sum(1 for ng in hyp_ngrams if ng in ref_ngrams)
            precision = matches / len(hyp_ngrams) if hyp_ngrams else 0
            
            # 简化的 BLEU 计算
            bleu_scores.append(precision)
        
        return {
            'bleu_2': np.mean(bleu_scores) if bleu_scores else 0,
            'num_samples': len(references)
        }
    
    def evaluate_code(self, generated_codes: List[str], test_cases: List[Dict]) -> Dict[str, float]:
        """评估代码生成"""
        pass_rates = []
        
        for code, tests in zip(generated_codes, test_cases):
            # 简化：检查代码是否包含预期结构
            # 实际应执行测试用例
            if 'def ' in code and 'return ' in code:
                pass_rates.append(1)
            else:
                pass_rates.append(0)
        
        return {
            'pass_rate': np.mean(pass_rates) if pass_rates else 0,
            'num_samples': len(generated_codes)
        }
    
    def run_benchmark(self, model_name: str, benchmark_name: str, 
                      task_func, data: List[Dict]) -> EvaluationResult:
        """运行基准评测"""
        predictions = []
        labels = []
        
        for item in data:
            # 运行模型预测
            pred = task_func(item['input'])
            predictions.append(pred)
            labels.append(item['label'])
        
        # 计算指标
        metrics = self.evaluate_accuracy(predictions, labels)
        
        # 记录结果
        result = EvaluationResult(
            model_name=model_name,
            benchmark=benchmark_name,
            metrics=metrics,
            details={
                'num_samples': len(data),
                'predictions': predictions[:10],  # 前 10 个预测
                'labels': labels[:10]
            }
        )
        
        self.results.append(result)
        return result
    
    def compare_models(self, results: List[EvaluationResult]) -> Dict:
        """比较多个模型"""
        comparison = defaultdict(dict)
        
        for result in results:
            for metric, value in result.metrics.items():
                comparison[result.model_name][metric] = value
        
        return dict(comparison)
    
    def visualize_results(self, results: List[EvaluationResult]):
        """可视化评测结果"""
        models = [r.model_name for r in results]
        metrics = results[0].metrics.keys() if results else []
        
        fig, axes = plt.subplots(1, len(metrics), figsize=(5*len(metrics), 4))
        if len(metrics) == 1:
            axes = [axes]
        
        for ax, metric in zip(axes, metrics):
            values = [r.metrics[metric] for r in results]
            bars = ax.bar(models, values)
            ax.set_title(metric)
            ax.set_ylabel('Score')
            ax.set_ylim(0, 1)
            
            # 添加数值标签
            for bar, value in zip(bars, values):
                height = bar.get_height()
                ax.text(bar.get_x() + bar.get_width()/2., height,
                       f'{value:.3f}', ha='center', va='bottom')
        
        plt.tight_layout()
        plt.savefig('evaluation_results.png')
        plt.show()

# 使用示例
if __name__ == "__main__":
    # 创建评测器
    evaluator = ModelEvaluator()
    
    # 模拟评测数据
    data = [
        {'input': '什么是机器学习？', 'label': 1},
        {'input': '深度学习是什么？', 'label': 1},
        {'input': 'Python 是什么？', 'label': 1},
        {'input': '无效问题', 'label': 0},
    ]
    
    # 模拟模型预测函数
    def model_a_prediction(question):
        # 简单规则：如果包含"什么"或"？"，预测为相关
        return 1 if '什么' in question or '？' in question else 0
    
    def model_b_prediction(question):
        # 随机预测
        import random
        return random.randint(0, 1)
    
    # 运行评测
    result_a = evaluator.run_benchmark("Model A", "QA_Benchmark", model_a_prediction, data)
    result_b = evaluator.run_benchmark("Model B", "QA_Benchmark", model_b_prediction, data)
    
    # 比较结果
    comparison = evaluator.compare_models([result_a, result_b])
    print("模型比较结果：")
    for model, metrics in comparison.items():
        print(f"\n{model}:")
        for metric, value in metrics.items():
            print(f"  {metric}: {value:.3f}")
    
    # 可视化
    evaluator.visualize_results([result_a, result_b])
```

### 中文评测示例

```python
import json
from typing import List, Dict

class ChineseEvaluator:
    """中文模型评测器"""
    
    def __init__(self):
        self.benchmarks = {}
    
    def load_c_eval(self, data_path: str):
        """加载 C-Eval 数据"""
        # 模拟数据结构
        subjects = ['数学', '物理', '化学', '历史', '文学']
        data = []
        
        for subject in subjects:
            # 模拟题目
            for i in range(10):
                data.append({
                    'subject': subject,
                    'question': f'这是{subject}的第{i+1}个问题？',
                    'choices': ['A. 选项1', 'B. 选项2', 'C. 选项3', 'D. 选项4'],
                    'answer': 'A'
                })
        
        self.benchmarks['c_eval'] = data
        return data
    
    def evaluate_choice_accuracy(self, predictions: List[str], answers: List[str]) -> Dict[str, float]:
        """评估选择题准确率"""
        correct = sum(1 for pred, ans in zip(predictions, answers) if pred == ans)
        total = len(predictions)
        
        return {
            'accuracy': correct / total if total > 0 else 0,
            'correct': correct,
            'total': total
        }
    
    def evaluate_subject_wise(self, predictions: List[Dict]) -> Dict[str, Dict[str, float]]:
        """按学科评估"""
        subjects = {}
        
        for pred in predictions:
            subject = pred['subject']
            if subject not in subjects:
                subjects[subject] = {'correct': 0, 'total': 0}
            
            subjects[subject]['total'] += 1
            if pred['prediction'] == pred['answer']:
                subjects[subject]['correct'] += 1
        
        # 计算准确率
        results = {}
        for subject, stats in subjects.items():
            results[subject] = {
                'accuracy': stats['correct'] / stats['total'] if stats['total'] > 0 else 0,
                'correct': stats['correct'],
                'total': stats['total']
            }
        
        return results
    
    def generate_report(self, model_name: str, results: Dict[str, Dict]) -> str:
        """生成评测报告"""
        report = f"# {model_name} 评测报告\n\n"
        report += "## 总体表现\n"
        
        total_correct = sum(r['correct'] for r in results.values())
        total_questions = sum(r['total'] for r in results.values())
        overall_accuracy = total_correct / total_questions if total_questions > 0 else 0
        
        report += f"- 总题数：{total_questions}\n"
        report += f"- 正确数：{total_correct}\n"
        report += f"- 总准确率：{overall_accuracy:.2%}\n\n"
        
        report += "## 分学科表现\n"
        for subject, stats in results.items():
            report += f"- **{subject}**：{stats['accuracy']:.2%} ({stats['correct']}/{stats['total']})\n"
        
        return report

# 使用示例
if __name__ == "__main__":
    # 创建评测器
    evaluator = ChineseEvaluator()
    
    # 加载数据
    data = evaluator.load_c_eval("c_eval_data.json")
    print(f"加载了 {len(data)} 道题目")
    
    # 模拟模型预测
    predictions = []
    for item in data:
        # 简单规则：总是选 A
        predictions.append({
            'subject': item['subject'],
            'question': item['question'],
            'prediction': 'A',
            'answer': item['answer']
        })
    
    # 按学科评估
    results = evaluator.evaluate_subject_wise(predictions)
    
    # 生成报告
    report = evaluator.generate_report("Test Model", results)
    print(report)
```
