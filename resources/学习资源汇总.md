## 九、学习路径建议

### 初学者（1-3 个月）
1. 了解 LLM 基本概念（本知识库 01-基础入门）
2. 学习 Prompt Engineering
3. 尝试 API 调用（对话机器人项目）
4. 阅读经典论文（Transformer、BERT、GPT）

### 进阶者（3-6 个月）
1. 深入 Transformer 架构（02-核心原理）
2. 学习 RAG 技术（知识库问答项目）
3. 实践模型微调（LoRA/QLoRA 项目）
4. 探索 Agent 开发

### 高级（6-12 个月）
1. 多 Agent 系统（多 Agent 协作项目）
2. 模型训练全流程
3. 参与开源贡献
4. 关注前沿技术（04-进阶应用/04-前沿技术.md）
