# LLM 发展历程

## 学习目标
学完本节后，你将能够：
- 了解大语言模型从早期到现代的发展脉络
- 掌握关键里程碑事件和代表性模型
- 理解技术演进的内在驱动力
- 认识不同阶段的技术特点和突破

## 核心知识点

### 1. 早期探索（1950s-2010s）

**1.1 统计语言模型时代**
- **n-gram 模型**：基于马尔可夫假设，预测下一个词的概率
- **最大熵模型**：结合多种特征进行语言建模
- **局限性**：无法捕捉长距离依赖，数据稀疏问题严重

**1.2 神经网络语言模型**
- **Word2Vec (2013)**：首次将词表示为稠密向量
- **GloVe (2014)**：结合全局统计信息和局部上下文
- **RNN/LSTM**：处理序列数据，但训练困难

### 2. Transformer 时代（2017-2019）

**2.1 关键突破**
- **2017**："Attention Is All You Need" 论文发布
- **2018**：BERT 发布，开启预训练+微调范式
- **2019**：GPT-2 发布，展示生成式模型的潜力

**2.2 技术特点**
- 完全基于注意力机制，摒弃 RNN 结构
- 可并行计算，训练效率大幅提升
- 预训练+微调成为标准范式

### 3. 大模型爆发期（2020-2022）

**3.1 规模竞赛**
- **GPT-3 (2020)**：1750 亿参数，展示 few-shot 学习能力
- **Gopher (2021)**：2800 亿参数，系统性研究规模效应
- **PaLM (2022)**：5400 亿参数，进一步提升性能

**3.2 关键进展**
- **指令微调**：InstructGPT、ChatGPT 的核心
- **RLHF**：人类反馈强化学习
- **思维链（CoT）**：提升复杂推理能力

### 4. 开源与普及（2023-至今）

**4.1 开源模型崛起**
- **LLaMA 系列**：Meta 开源，推动社区发展
- **ChatGLM**：中文大模型开源代表
- **BLOOM**：多语言大模型

**4.2 效率优化**
- **量化技术**：降低推理资源需求
- **蒸馏技术**：知识传递到小模型
- **LoRA/QLoRA**：高效微调方法

```
   ↓

   ↓

   ↓
2020  GPT-3 - 1750亿参数，Few-shot
   ↓

   ↓
2022  ChatGPT - RLHF + 对话
   ↓
```


### 模拟 n-gram 语言模型

```python
from collections import defaultdict, Counter
import random

class NGramModel:
    """简化版 n-gram 语言模型"""
    
    def __init__(self, n=2):
        self.n = n
        self.ngrams = defaultdict(Counter)
    
    def train(self, text):
        """训练模型"""
        words = text.lower().split()
        for i in range(len(words) - self.n + 1):
            context = tuple(words[i:i+self.n-1])
            next_word = words[i+self.n-1]
            self.ngrams[context][next_word] += 1
    
    def generate(self, start_words, length=20):
        """生成文本"""
        result = list(start_words)
        context = tuple(start_words[-(self.n-1):])
        
        for _ in range(length):
            if context in self.ngrams:
                # 根据概率选择下一个词
                candidates = self.ngrams[context]
                total = sum(candidates.values())
                r = random.randint(1, total)
                cumulative = 0
                for word, count in candidates.items():
                    cumulative += count
                    if cumulative >= r:
                        next_word = word
                        break
            else:
                next_word = random.choice(list(set().union(*[c.keys() for c in self.ngrams.values()])))
            
            result.append(next_word)
            context = tuple(result[-(self.n-1):])
        
        return ' '.join(result)

# 使用示例
text = "the quick brown fox jumps over the lazy dog the quick brown fox is very quick"
model = NGramModel(n=2)
model.train(text)

print("生成示例：", model.generate(["the", "quick"], length=10))
```
