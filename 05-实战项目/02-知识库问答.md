# 实战项目二：知识库问答系统（RAG）

> 构建基于检索增强生成（RAG）的智能问答系统，让 LLM 能够基于你的私有知识库回答问题。

## 项目目标

实现一个完整的 RAG 系统，能够：

1. **文档处理** — 支持 PDF、Markdown、Word 等格式
2. **向量存储** — 高效存储和检索文档向量
3. **语义检索** — 基于语义相似度找到相关内容
4. **智能问答** — 结合检索结果生成准确答案
5. **来源追踪** — 显示答案的引用来源

### 1. 文档处理

```python
from pathlib import Path
from PyPDF2 import PdfReader

def load_documents(doc_dir: str) -> list[dict]:
    """加载目录下的所有文档"""
    docs = []
    
    for file_path in Path(doc_dir).rglob("*"):
        if file_path.suffix == ".pdf":
            text = extract_pdf(file_path)
        elif file_path.suffix == ".md":
            text = file_path.read_text(encoding="utf-8")
        elif file_path.suffix == ".txt":
            text = file_path.read_text(encoding="utf-8")
        else:
            continue
        
        docs.append({
            "content": text,
            "source": str(file_path),
            "filename": file_path.name
        })
    
    return docs

def extract_pdf(file_path: Path) -> str:
    """提取 PDF 文本"""
    reader = PdfReader(str(file_path))
    text = ""
    for page in reader.pages:
        text += page.extract_text() + "\n"
    return text
```

### 2. 文本分块

```python
def chunk_text(text: str, chunk_size: int = 500, overlap: int = 50) -> list[str]:
    """将文本分块"""
    chunks = []
    start = 0
    
    while start < len(text):
        end = start + chunk_size
        chunk = text[start:end]
        
        # 在句号处截断，避免切断句子
        if end < len(text):
            last_period = chunk.rfind("。")
            if last_period > chunk_size * 0.5:
                chunk = chunk[:last_period + 1]
                end = start + last_period + 1
        
        chunks.append(chunk)
        start = end - overlap
    
    return chunks
```

### 3. 向量化与存储

```python
import chromadb
from openai import OpenAI

class VectorStore:
    def __init__(self, collection_name="knowledge_base"):
        self.client = chromadb.PersistentClient("./chroma_db")
        self.collection = self.client.get_or_create_collection(
            name=collection_name,
            metadata={"hnsw:space": "cosine"}
        )
        self.openai = OpenAI()
    
    def embed(self, texts: list[str]) -> list[list[float]]:
        """生成文本向量"""
        response = self.openai.embeddings.create(
            model="text-embedding-3-small",
            input=texts
        )
        return [d.embedding for d in response.data]
    
    def add_documents(self, docs: list[dict]):
        """批量添加文档"""
        for doc in docs:
            chunks = chunk_text(doc["content"])
            embeddings = self.embed(chunks)
            
            ids = [f"{doc['source']}_{i}" for i in range(len(chunks))]
            metadatas = [
                {"source": doc["source"], "chunk": i}
                for i in range(len(chunks))
            ]
            
            self.collection.add(
                ids=ids,
                embeddings=embeddings,
                documents=chunks,
                metadatas=metadatas
            )
    
    def search(self, query: str, top_k: int = 5) -> list[dict]:
        """语义检索"""
        query_embedding = self.embed([query])[0]
        
        results = self.collection.query(
            query_embeddings=[query_embedding],
            n_results=top_k
        )
        
        return [
            {
                "content": doc,
                "source": meta["source"],
                "distance": dist
            }
            for doc, meta, dist in zip(
                results["documents"][0],
                results["metadatas"][0],
                results["distances"][0]
            )
        ]
```

### 4. RAG 问答链

```python
class RAGSystem:
    def __init__(self, vector_store: VectorStore, model: str = "gpt-4"):
        self.vs = vector_store
        self.client = OpenAI()
        self.model = model
    
    def ask(self, question: str, top_k: int = 3) -> dict:
        """RAG 问答"""
        # 1. 检索相关内容
        results = self.vs.search(question, top_k)
        context = "\n\n".join([r["content"] for r in results])
        
        # 2. 构建提示词
        prompt = f"""基于以下上下文回答问题。如果上下文中没有相关信息，请说明。

上下文：
{context}

问题：{question}

请提供准确、简洁的回答，并在最后列出引用来源。"""
        
        # 3. 生成答案
        response = self.client.chat.completions.create(
            model=self.model,
            messages=[
                {"role": "system", "content": "你是一个知识库问答助手，基于提供的上下文回答问题。"},
                {"role": "user", "content": prompt}
            ]
        )
        
        answer = response.choices[0].message.content
        
        return {
            "question": question,
            "answer": answer,
            "sources": [
                {"source": r["source"], "content_preview": r["content"][:100]}
                for r in results
            ]
        }
```

### 2. 重排序（Re-ranking）

使用专门的重排序模型提升检索质量：

```python
from sentence_transformers import CrossEncoder

reranker = CrossEncoder("cross-encoder/ms-marco-MiniLM-L-6-v2")

def rerank(query: str, candidates: list[dict]) -> list[dict]:
    pairs = [(query, c["content"]) for c in candidates]
    scores = reranker.predict(pairs)
    
    # 按分数重新排序
    for c, s in zip(candidates, scores):
        c["rerank_score"] = s
    
    return sorted(candidates, key=lambda x: x["rerank_score"], reverse=True)
```

## 常见问题

### Q: 检索不到相关内容怎么办？
- 检查文档是否正确加载和分块
- 尝试降低检索阈值
- 使用查询扩展技术
- 优化分块策略（大小、重叠）

### Q: 答案引用错误？
- 检查元数据是否正确存储
- 使用重排序提升相关性
- 在提示词中强调"基于上下文回答"

### Q: 如何处理多语言文档？
- 使用多语言 Embedding 模型（如 bge-m3）
- 或为不同语言维护不同的集合
